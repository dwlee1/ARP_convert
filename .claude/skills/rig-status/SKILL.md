name: rig-status
description: ARP 리깅 변환 현황 보고
---

프로젝트의 리깅 파일들을 분석하여 AutoRig Pro(ARP) 변환 현황을 보고합니다.

1. 프로젝트 전체에서 리그 관련 파일을 검색합니다 (`_rig` 포함 .blend 파일).
2. ARP 관련 파일을 식별합니다 (`arp_` 접두사 .py 파일, ARP 관련 이름 패턴).
3. 다음 현황을 정리합니다:
   - 전체 리그 파일 수
   - ARP 적용 추정 파일 수
   - 미변환 추정 파일 수
   - 폴더별/동물별 변환 상태
4. 결과를 표 형태로 요약합니다.

검색 대상:
- `**/*rig*.blend` — 리그 파일
- `**/arp_*.py` — ARP 레퍼런스 데이터
- `blend_files_report.csv` — 기존 스캔 결과 참고

참고: .blend 파일 내부의 리깅 방식은 Blender에서만 확인 가능합니다. 이 스킬은 파일명과 구조 기반으로 추정합니다.
